Глагол "быть" - sein [зайн]

На русском На немецком Транскрипция
Я есть Ich bin [их бин]
Ты есть Du bist [ду бист]
Он есть Er ist [эа ист]
Она есть Sie ist [зи ист]
Оно есть Es ist [эс ист]
Мы есть Wir sind [виа зинд]
Вы(множ) есть Ihr seid [иа зайд]
Они есть Sie sind [зи зинд]
Вы(уваж) есть Sie sind [зи зинд]
